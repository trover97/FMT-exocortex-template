# Подключение IWE из браузера за 2 минуты

> **Для кого:** Ты хочешь подключить ИИ-помощника, который знает методологию МИМ, знает тебя и растёт вместе с тобой. Не нужно ничего устанавливать — работает прямо в браузере.
>
> **Что нужно:** Подписка Claude Pro ($20/мес) на claude.ai. Это даёт доступ к ИИ-помощнику. Знания МИМ подключаются бесплатно на 30 дней (потом ~$10/мес на system-school.ru).

---

## Шаг 1. Открой настройки Claude

Зайди на [claude.ai](https://claude.ai) → нажми на имя профиля (левый нижний угол) → **Settings**

## Шаг 2. Добавь IWE

В настройках найди раздел **Integrations** → нажми **Add custom integration** (или **Add connector**):

- **Name:** `IWE`
- **URL:** `https://mcp.aisystant.com/mcp`

Нажми **Save**.

## Шаг 3. Авторизуйся

После добавления Claude запросит вход — войди через аккаунт на system-school.ru.

Если аккаунта нет — зарегистрируйся на [system-school.ru](https://system-school.ru). Первые 30 дней бесплатно.

## Шаг 4. Готово — начни разговор

Открой новый чат и напиши:

> **С чего мне начать?**

ИИ-помощник загрузит контекст IWE и проведёт тебя дальше: предложит задать вопрос, пройти диагностику уровня или получить маршрут развития.

---

## Что ты получаешь

| Возможность | Что делает |
|-------------|-----------|
| **База знаний** | Поиск по 5000+ документам МИМ — методологии, руководства, курсы |
| **Твой профиль** | Цели, уровень, прогресс — ИИ знает тебя и адаптирует ответы |
| **Навигатор** | Подскажет с чего начать, какую программу выбрать, как выстроить ритм |
| **Диагност** | Определит твою ступень развития и что мешает расти |
| **Личная база знаний** | Сохранение заметок, идей, стратегии — ИИ предложит создать при первом разговоре |

## Зачем это нужно (коротко)

Голый ChatGPT или Claude — это умный новичок в первый рабочий день. Много знает, но не знает *тебя*. IWE превращает его в персонального помощника:

- **Знает предмет** — за ним база знаний по системному мышлению, менеджменту, инженерии
- **Знает тебя** — твои цели, уровень, контекст
- **Растёт с тобой** — чем больше работаешь, тем точнее помощь

Конкуренция сейчас не между людьми, а между людьми вместе с их ИИ-помощниками.

## Другие AI-клиенты

IWE работает не только в Claude.ai:

| Клиент | Как подключить |
|--------|---------------|
| **Cursor** | Settings → Features → MCP → Add New MCP Server → URL: `https://mcp.aisystant.com/mcp` |
| **ChatGPT** | Business/Enterprise → Workspace Settings → Add MCP connector |
| **Claude Code (полный IWE)** | [SETUP-GUIDE.md](../SETUP-GUIDE.md) — планирование дня, ИИ-стратег, автоматические отчёты |

## Проблемы

| Проблема | Решение |
|----------|---------|
| Нет раздела Integrations в Claude | Нужна подписка Claude Pro ($20/мес) |
| Ошибка при авторизации | Проверь что есть аккаунт на system-school.ru |
| Claude не использует IWE-инструменты | Открой новый чат. Напиши: «Используй IWE — найди документы про системное мышление» |
| «Не нашёл в базе знаний» | Проверь что интеграция активна: Settings → Integrations → IWE → Connected |
