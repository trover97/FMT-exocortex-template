# Lifework — личные стратегические документы

> Здесь хранятся **твои** заполненные lifework-документы.
> Шаблоны для создания — в `PACK-personal/.../lifework-templates/`.
> Метод составления — `PD.METHOD.018` (PACK-personal).

## Структура по уровням охвата (PD.FORM.090)

| Уровень | Инвариантный документ | Рекомендуемое имя файла |
|---------|----------------------|------------------------|
| 1. Я | Личный устав | `1. Личный устав.md` |
| 2. Ближний круг | Семейный уклад | `2. Семейный уклад.md` |
| 3. Дальний круг | Профессиональные принципы | `3. Профессиональные принципы.md` |
| 4. Сообщество | Манифест | `4. Манифест <название>.md` |
| 5. Общество | Миссия | `5. Моя миссия.md` |
| 6. Человечество | Конституция Дела | `6. Конституция Дела.md` |

## Как создавать

1. Возьми шаблон из `PACK-personal/pack/personal-development/04-work-products/lifework-templates/`
2. Создай здесь файл по рекомендуемому имени
3. Стратег читает эти файлы на сессии (шаг 5b «Линза калибра») для измерения C_act

## Карта документов

Создай `0. Карта документов.md` — навигационный индекс с кратким описанием каждого документа и его статусом (draft/active). Файл с нулевым номером — точка входа в папку.
