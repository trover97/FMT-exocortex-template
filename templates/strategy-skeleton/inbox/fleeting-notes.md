# fleeting-notes.md — входящие заметки

> **Что это.** Однострочные заметки, идеи, ссылки, замечания, которые ещё не доросли до РП или знания в Pack. Аналог Inbox.

## Правила

1. **Свежие сверху.** Каждая запись с датой `YYYY-MM-DD HH:MM`.
2. **Маркеры маршрутизации:** `[ke]` — кандидат в Knowledge Extraction (в Pack), `[wp]` — кандидат в РП, `[idea]` — просто идея, `[link]` — ссылка.
3. **Триаж при Day Close** — `/day-close` подсвечивает накопившиеся записи и предлагает маршрутизировать.
4. **Архив:** обработанные записи → `archive/fleeting-notes-YYYY-MM.md` (опционально).

## Записи

```
2026-MM-DD HH:MM [idea] _пример: попробовать читать книги через TTS на прогулке_
2026-MM-DD HH:MM [ke] _пример: выяснил различение «роль ≠ исполнитель» — добавить в Pack_
2026-MM-DD HH:MM [wp] _пример: автоматизировать утренний дайджест_
```
