# Step 06c: План — финализация

## ВДВ

| | |
|---|---|
| **Вход** | Бюджет + ТОС (шаг 06b), список кандидатов |
| **Действие** | Сверка РП↔НЭП, финализация таблицы WeekPlan |
| **Выход** | Обновлённая таблица РП в WeekPlan |

## Инструкция для Claude

1. **Сверка РП↔НЭП** — для каждого выбранного РП назови источник:
   - `R{N}` — приоритет месяца
   - `S-НЭП-X` — неудовлетворённость
   - `carry` — carry-over
   - РП без источника — сигнал «зачем это делаем?», требует решения сейчас

2. **Финализация таблицы:**
   Обнови таблицу РП в WeekPlan с колонками:
   `🚦 / # / РП / Бюджет / Источник / Статус / Дедлайн / Репо`

3. **Вопрос пилоту:**
   > «Продолжить к нерегулярным блокам / скорректировать / пропустить?»
