# Step 03: Неудовлетворённости (НЭП)

## ВДВ

| | |
|---|---|
| **Вход** | `docs/Dissatisfactions.md` + новые сигналы из inbox (шаг 02) и нерегулярных блоков (шаг 07) |
| **Действие** | Закрыть разрешённые НЭП, обсудить новые, обновить Dissatisfactions.md |
| **Выход** | Актуализированный Dissatisfactions.md, решения по новым НЭП |

## Инструкция для Claude

1. **Прочитать** `docs/Dissatisfactions.md`

2. **Проверить:**
   - Какие операционные НЭП разрешены (можно закрыть)?
   - Есть ли стратегические НЭП без привязки к РП на этой неделе?
   - Новые НЭП из inbox / нерегулярных блоков

3. **Обсудить с пилотом:**
   - Закрыть разрешённые
   - Сформулировать новые
   - Проверить привязку к РП

4. **Обновить** `Dissatisfactions.md` по решениям пилота

5. **Вопрос пилоту:**
   > «Продолжить к стратегической сверке / скорректировать / пропустить?»
