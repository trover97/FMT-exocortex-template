# Step 06b: План — бюджет + ТОС

## ВДВ

| | |
|---|---|
| **Вход** | Candidate pool с приоритетами (шаг 06a) |
| **Действие** | Распределить бюджет, определить ТОС недели, зафиксировать «что НЕ берём» |
| **Выход** | Утверждённый бюджет, ТОС недели, explicit «не берём» |

## Инструкция для Claude

1. **Budget fit + Time-boxing** — пилот корректирует:
   - Сумма бюджетов ≤ доступные часы (превышение ≤10%)
   - Каждый РП имеет hard cap: ≤4h маленький, 4-15h большой, open-ended запрещены

2. **ТОС недели** — пилот утверждает ОДНО ограничение, снятие которого разблокирует наибольшую пропускную способность. Запиши в шапку WeekPlan.

3. **Explicit «не берём»:**
   - Зафиксируй явный список «Что НЕ берём в эту неделю и почему»
   - Это предотвращает implicit carry-over

4. **Вопрос пилоту:**
   > «Продолжить к финализации / скорректировать / пропустить?»
